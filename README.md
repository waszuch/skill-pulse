# SkillPulse

Personal skill and project portfolio tracker with real-time synchronization and authentication. Built with Next.js 16, TypeScript, Convex, Clerk, and shadcn/ui.

## Features

- Add, edit, and delete skills with proficiency levels (1-5) and custom tags
- Showcase projects with descriptions, tech stacks, and live links
- Visual analytics with interactive charts (skill distribution, top technologies)
- Real-time data synchronization across devices
- User authentication (email/password, OAuth)
- Database persistence with Convex
- Dark/light mode toggle
- Form validation with TanStack Form and Zod
- Smooth animations and transitions
- Fully responsive design

## Development

```bash
pnpm dev
```

Open http://localhost:3000

## Build

```bash
pnpm build
pnpm start
```
